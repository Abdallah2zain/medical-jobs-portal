# ملاحظات اختبار الموقع

## صفحة طلب التوظيف (/apply)
- الصفحة تعمل بشكل صحيح
- نموذج الخطوات (Stepper) يظهر بشكل صحيح: المدينة → الوظيفة → بياناتك → تم الإرسال
- قائمة اختيار المدينة موجودة
- ملاحظة: القائمة تحتاج إلى بيانات من قاعدة البيانات لعرض المدن المتاحة

## الحالة
- الواجهة تعمل بشكل صحيح
- يجب إضافة بيانات تجريبية للمدن والوظائف في قاعدة البيانات
